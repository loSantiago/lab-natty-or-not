```markdown
# FAKE - Paper Científico Sobre Mosquitos Gigantes ;)

## 📒 Descrição
Este paper científico tem o intuito de mostrar a capacidade das IAs de criar conteúdo fake.
Este é um paper de poucas páginas, que mostra uma pesquisa absurda e ainda tem uma matéria.
Com uma áudio reportagem fake para reforçar tal conteúdo.

## 🤖 Tecnologias Utilizadas
CHATGPT MINI 4.0 -> Criação do texto inicial  

AIDOCMAKER -> Para adição do texto e criação do documento.

GEMINI -> Imagens dos mosquitos + Texto da Reportagem

TTSMAKE -> Narração gratuita.

SpeechGen -> Narração gratuita

## 🧐 Processo de Criação
O processo de criação foi realizado utilizando o uso da IA AIDOCMAKER para criar o paper fake,
após busquei o GEMINI para desenvolver uma narrativa de reportagem para divulgar o paper e
posteriormente busquei uma IA gratuita de narração, onde o TTSMAKER se mostrou excelente para
o propósito, se eu tivesse pagando por algo, o resultado seria infinitamente superior e mais
confiável. Sendo assim, podemos ver como pequenos ajustes nos conteúdos gerados por IA podem
ter uma taxa de alto convencimento de massas, analisando ainda o aspecto gratuito das ferramentas
que produziram o material.

## 🚀 Resultados
O resultado se encontra na pasta "DATA" neste mesmo repositório.

## 💭 Reflexão (Opcional)
Apesar de o texto ter fontes reais e um embasamento científico em certos pontos,
Não se trata de uma preocupação de saúde ou epidemia eminente no momento.
A criação deste conteúdo foi extremamente fácil, apenas usando prompts simples 
e sites específicos de IA gratuita.
```
